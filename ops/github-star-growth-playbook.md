# GitHub Star Growth Playbook

This playbook uses legitimate, community-friendly growth tactics. Do not buy stars, use bots, run fake accounts, or trade stars.

## Positioning

One sentence:

> Two ClawHub skills for AI-assisted QA: code risk review and requirement-test-code traceability.

Target audiences:

- QA engineers and SDETs
- engineering managers
- AI app builders
- agent/RAG/LLM workflow builders
- release managers
- open-source maintainers

## Star Growth Actions

### 1. Make the repo easy to understand

- Keep the README value proposition in the first 10 seconds.
- Include install commands above the fold.
- Link both ClawHub listings near the top.
- Add screenshots or sample reports when available.
- Keep examples copy-pasteable.

### 2. Create useful content

Publish short posts around:

- "How to turn Git Diff into a release-quality gate"
- "Requirement-test-code traceability for AI products"
- "Testing Agent/RAG workflows without vague checklists"
- "Coverage gap detection before release"

### 3. Launch in relevant communities

Share when there is a concrete example or release note:

- GitHub Discussions and issues in relevant QA/AI repos when genuinely helpful
- Hacker News Show HN
- Reddit: r/QualityAssurance, r/softwaretesting, r/LocalLLaMA when relevant
- LinkedIn engineering/QA posts
- X/Twitter threads with example reports
- Team QA and engineering communities

### 4. Use release cadence

Ship small visible releases:

- new prompt examples;
- new report templates;
- new Agent/RAG testing patterns;
- sample reports;
- compatibility notes.

### 5. Cross-link ClawHub and GitHub

- GitHub README links to ClawHub install pages.
- ClawHub README links to GitHub source and issue tracker.
- GitHub releases mention ClawHub versions.
- ClawHub changelog mentions GitHub release notes.

## Weekly Metrics

Track:

- GitHub stars
- GitHub clones/views
- ClawHub downloads/installs if available
- ClawHub search rank for target keywords
- README click-through from GitHub to ClawHub
- Issues/discussions opened
