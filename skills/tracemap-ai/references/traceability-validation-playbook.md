# Traceability Validation Playbook

## Coverage Categories

- Fully covered
- Partially covered
- Uncovered
- Redundant test case
- Blocked by missing evidence
- Untestable or ambiguous requirement

## Mapping Dimensions

Map every requirement to:

- acceptance criteria;
- test case or test gap;
- code location or implementation gap;
- data fixture or mocked condition;
- expected evidence;
- execution result when available.
