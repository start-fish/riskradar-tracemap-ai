# RiskRadar AI

AI Code Review and Code Risk Review quality gate for software teams.

RiskRadar AI reviews Git Diff and source code for functional risk, runtime failure paths, dependency impact, security exposure, performance risk, AI/Agent/RAG workflow risk, and regression-test gaps. It produces a release-ready Markdown report with severity, exact code location, impact, evidence, and test strategy.

## Why Star This Skill

- Works across backend, frontend, mobile, API, platform, data, and AI systems.
- Finds risks in LLM, Agent, RAG, prompt, tool-calling, model-routing, and fallback workflows.
- Converts code review into actionable release-gate evidence.
- Produces specific tests: unit, integration, E2E, adversarial, resilience, and monitoring.
- Helps teams reduce missed defects before release.
