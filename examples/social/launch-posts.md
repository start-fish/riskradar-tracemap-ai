Show HN: RiskRadar + TraceMap — ClawHub skills for AI-assisted QA

I built two ClawHub skills for release-quality workflows:

- RiskRadar AI reviews Git Diff and source changes for release risks, runtime failures, security exposure, dependency impact, and regression-test gaps.
- TraceMap AI maps requirements, tests, and code changes into a traceability matrix and highlights coverage gaps.

The goal is not generic AI advice, but reusable QA evidence that teams can inspect before shipping.
