# TraceMap AI Prompt Examples

## PRD Coverage Review

```text
Use TraceMap AI to map this PRD, acceptance criteria, test cases, and code changes.
Build a requirement-test-code traceability matrix and highlight coverage gaps before release.
```

## Test Case Validation

```text
Use TraceMap AI to validate these test cases against the product requirements.
Identify missing positive cases, negative cases, boundary cases, compatibility checks, and release acceptance evidence.
```

## Multi-Repo Release Sign-Off

```text
Use TraceMap AI to review this multi-repo release.
Map each requirement to changed services, test evidence, owners, and unresolved release risks.
```
