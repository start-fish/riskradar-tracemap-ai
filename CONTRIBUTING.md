# Contributing

Thanks for helping improve RiskRadar AI and TraceMap AI.

## High-Value Contributions

- Add realistic Git Diff review prompts.
- Add sample risk reports for backend, frontend, mobile, data, or AI/Agent/RAG systems.
- Add PRD/test-case traceability examples.
- Improve release-quality gate checklists.
- Report unclear instructions or missing coverage in either skill.

## Before Opening a PR

1. Keep examples generic and safe to publish.
2. Remove company names, secrets, customer data, internal URLs, and private code.
3. Prefer concise examples that users can copy into ClawHub.
4. Explain which workflow the example improves.

## Issue Types

Open an issue for:

- skill feedback;
- missing prompt examples;
- confusing output structure;
- incorrect or risky recommendations;
- new QA workflow ideas.
